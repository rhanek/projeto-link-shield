# \## Sobre o Projeto

# 

# O \*Link Shield\* é uma aplicação web que permite analisar qualquer URL e gerar um relatório completo de segurança. O projeto verifica protocolo HTTPS, analisa o domínio em busca de padrões suspeitos e consulta a API do Google Safe Browsing para identificar possíveis ameaças.

# 

# Ideal para demonstrar conhecimentos em consumo de APIs, manipulação de DOM, JavaScript moderno e design responsivo.

# 

# \## Funcionalidades

# 

# \- ✅ Verificação de protocolo HTTPS

# \- ✅ Análise de domínio (IPs, caracteres suspeitos, subdomínios)

# \- ✅ Consulta à \*Google Safe Browsing API\*

# \- ✅ Sistema de pontuação de segurança (0-100) com barra de progresso animada

# \- ✅ Histórico de consultas salvas no LocalStorage

# \- ✅ Exportação do relatório em arquivo .txt

# \- ✅ Interface moderna, clean e totalmente responsiva

# 

# \## Tecnologias Utilizadas

# 

# \- \*HTML5\* Semântico

# \- \*CSS3\* (Flexbox + animações)

# \- \*JavaScript (ES6+)\* 

# \- \*Fetch API\*

# \- \*Google Safe Browsing API\*
